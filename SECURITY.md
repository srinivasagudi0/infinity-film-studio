# Security Policy

## Supported versions
The latest commit on `main` is supported. Older snapshots may not receive fixes.

## Reporting a vulnerability
- Email: [security@infinityfilmstudio.com](mailto:security@infinityfilmstudio.com)
- Optional: mark the issue as **confidential** by sending a private GitHub Security Advisory if your org has access.

Please include:
- Steps to reproduce or proof of concept
- Impact and severity
- Any suggested mitigations

## Response timeline
- Acknowledge receipt: within 3 business days
- Initial assessment: within 7 business days
- Fix or advisory: communicated as soon as validated, typically within 30 days

We greatly appreciate responsible disclosure and will credit reporters unless anonymity is requested.
