# Contributing to Infinity Film Studio

Thanks for your interest! Please read `docs/CONTRIBUTING.md` for the ground rules, coding standards, and testing expectations before opening a PR.

Quick checklist:
- Fork the repo and branch off `main`
- Add or update tests under `backend/tests`
- Run `pytest -q` and `npm run build` before pushing
- Follow the [Code of Conduct](CODE_OF_CONDUCT.md)
- Use the PR template and keep commits focused
